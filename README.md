SMF::Solaris
============

A Ruby interface to the Service Management Facility (SMF) in Solaris and Illumos

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'smf-solaris'
```

And then execute:

```bash
bundle
```

Or install it yourself as:

```bash
gem install smf-solaris
```
